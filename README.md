# ATH Website

This repository contains the code for the ATH Website, including the store page with Google Sheets integration and PayPal checkout.

## Main Files

- `index.html` - Main landing page
- `store.html` - Store page with Google Sheets integration and PayPal checkout
- `commands.html` - Commands page
- `discord.html` - Discord page
- `contact.html` - Contact page
- `new-wagers-bay.html` - Wagers Bay page

## JavaScript Files

- `sheets-inventory.js` - Google Sheets integration for inventory management
- `inventory.js` - Local inventory management
- `script.js` - General site functionality
- `image-locations.js` - Image location management

## Documentation

For detailed information about the website's features and setup instructions, please refer to the [documentation](documentation.md) file, which covers:

1. Google Sheets Integration
2. PayPal Integration
3. Image System
4. Development Notes

## Google Apps Script

The `google-apps-script.js` file contains the Google Apps Script code for updating inventory in Google Sheets after purchases. See the documentation for setup instructions.

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser to view the site
3. Follow the setup instructions in the documentation to configure Google Sheets and PayPal integration

## Development

The site uses vanilla HTML, CSS, and JavaScript without any frameworks or build tools. To make changes:

1. Edit the HTML, CSS, or JavaScript files
2. Refresh the page in your browser to see your changes

## Debugging

For debugging purposes, a hidden debug panel is included in `store.html` that can be toggled with Ctrl+Shift+D.
